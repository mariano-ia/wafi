# Wafi

**Fidelización simple. Clientes que vuelven.**

Wafi reemplaza la tarjeta de sellos de cartón de los locales gastronómicos por **sellos digitales**: el cliente escanea un QR, suma sellos en su celular (Apple/Google Wallet + PWA) y canjea premios. El comercio lo gestiona desde un dashboard web y paga la suscripción con **Mercado Pago**. Pensado para **Argentina**, foco en **cafeterías**.
